# Sentinels Landing Page

This is the official landing page for the **Sentinels Autonomous Security Auditor**, built for the 2026 Agentic Security Hackathon.

## Features
- **Premium SOC Aesthetic**: Dark, high-density UI with glassmorphism and glowing accents.
- **Autonomous Workflow Visualization**: Detailed breakdown of the Sentinel analysis chain.
- **Live Audit Logs**: Real-time terminal simulation showing actual scan data.
- **Agentic Commerce Integration**: Highlighting USDC-based micro-transactions and bounty automation.
- **Four Horsemen Orchestration**: Insight into the multi-model intelligence backend.

## Tech Stack
- **HTML5/Tailwind CSS**: Core structure and styling.
- **Lucide Icons**: Premium iconography.
- **Vanilla JS**: Micro-animations and terminal simulation.

## Deployment
This is a single-page static site. To deploy:
1. Push to a GitHub repository.
2. Enable GitHub Pages on the `main` branch.
3. Access the site via the provided GitHub Pages URL.
